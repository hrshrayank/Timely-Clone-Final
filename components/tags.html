Tags

Tags let you standardize time reporting across your workspace. Add tags for every activity you want to track here, then “allow” the ones you need on individual projects. Create tag lists to define the
most appropriate tags for specific project work.

We’re streamlining Tags!

In the coming weeks, we’ll be tweaking Tags to make it easier to manage large numbers of tags and projects. The first tweak will enable all tags on a tag list whenever the latter is added to a project. It should make it much easier to add and remove tags across tens (or thousands!) of projects. Archiving tags and more to follow!
